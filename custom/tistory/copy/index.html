table 태그 설명에 대한 표
태그명 속성 태그의 의미 및 특징
<table></table>

1. 표를 만들 때 사용하는 태그, 행(Row)과 열(Column)의 2차원 정보로 구성

2. 자식 요소로 반드시 tr 태그를 정의, table -> tr -> th or td 순으로 마크업해야 함
<tr></tr> tr 태그는 table 태그의 자식 요소로 표에서 행(Row)을 정의할 때 사용
<th></th>

colspan

1. 합치기 될 열(Column)의 수 지정

2. 만약 3개의 열을 합친다면 <td colspan="3">로 정의

rowspan

1. 합치기 될 행(Row)의 수 지정

2. 만약 3개의 행을 합친다면 <td rowspan="3">로 정의

 th 태그는 제목 역할을 하는 셀을 정의할 때 사용
<td></td> td 태그는 tr 태그의 자식 요소로 표에서 열(Column)을 정의할 때 사용
